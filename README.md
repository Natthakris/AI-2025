## สะท้อนคิด
**สิ่งที่ได้เรียนรู้**

- ทักษะทางเทคนิค: เช่น การเตรียมข้อมูล, การเลือกโมเดล, การปรับพารามิเตอร์ และการประเมินผล
- การคิดเชิงระบบ: มองเห็นความเชื่อมโยงระหว่างข้อมูล, อัลกอริทึม และการนำไปใช้จริง
- การสื่อสารผลลัพธ์: การอธิบายผลลัพธ์ให้คนที่ไม่ใช่สายเทคนิคเข้าใจได้

**ความท้าทายที่พบ**
  
- คุณภาพข้อมูล: ข้อมูลที่ไม่สะอาดหรือไม่ครบถ้วนส่งผลต่อความแม่นยำของโมเดล
- การตีความผลลัพธ์: ไม่ใช่แค่ได้ตัวเลข แต่ต้องเข้าใจว่ามันสะท้อนอะไรในโลกจริง
- การจัดการเวลาและทรัพยากร: การทดลองโมเดลหลายแบบใช้เวลาและพลังงานมาก

**สิ่งที่พัฒนาได้ต่อไป**
  
- การออกแบบ workflow ที่ชัดเจนตั้งแต่ต้นจนจบ
- การใช้ visualization เพื่อช่วยให้ผลลัพธ์เข้าใจง่ายขึ้น
- การเชื่อมโยงงานวิจัยหรือทฤษฎีเข้ากับการทดลองจริง เพื่อเพิ่มความน่าเชื่อถือ
